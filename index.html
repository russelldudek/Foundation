The real mandate

Build the operating layer between executive intent and technical capacity.

The title says operations. The work is executive extension: keep Scout, tagging, analytics, data, marketing, and partnerships aligned to customer decisions, economics, ownership, and the next explicit choice.

120-day entry

Earn context. Expose load. Install cadence.

The first four months use tactical empathy before prescription—learn the system, contract the portfolio, prove one leverage point, then scale the rhythm.

Days 1–30

Read the load

Map customer decisions, active initiatives, hidden work, decision rights, dependencies, and existing cadence.

Days 31–60

Install the minimum system

Create initiative contracts, a weekly load review, and an executive portfolio view.

Days 61–90

Prove one leverage point

Release measurable capacity through one AI or automation pilot with adoption and trust controls.

Days 91–120

Scale the cadence

Institutionalize quarterly planning and a decision-ready operating review.
